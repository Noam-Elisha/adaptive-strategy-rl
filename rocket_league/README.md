# Rocket League Strategy Bot

Stage 2 of the Adaptive Strategy RL project. This module will apply the
strategy-conditioned training approach (proven in the proof of concept) to an
actual Rocket League bot using the
[GigaLearn](https://github.com/ZealanL/GigaLearnCPP-Leak) framework with PPO.
